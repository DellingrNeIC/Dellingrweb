## Getting resources via Dellingr

Dellingr offers a self-service for discovering and applying for resources from a number of HPC data centers.
At this moment in time we offer batch resources only with cloud services in the pipeline.

### Applying for resource

#### Register in Dellingr self-service

1. Go to [share.neic.no](https://share.neic.no).

2. Login using your EduGain identity.

<img class="img-responsive center-block" src="/assets/img/guide/login.png" width="350">

#### Create your organization (optional)

If you are a Principal Investigator (PI), you should first create an organization.

<img class="img-responsive center-block" src="/assets/img/guide/new_organization.png" width="700">

#### Finding a matching resource

1. Login into [share.neic.no](https://share.neic.no) and select organization you represent.
2. Pick Marketplace from the menu.
<img class="img-responsive center-block" src="/assets/img/guide/marketplace.png" width="700">
3. Browse, compare and discover available resources.

#### Requesting access

1. Create a project for which you'd like to get resources via a project tab.
<img class="img-responsive center-block" src="/assets/img/guide/project.png" width="700">

2. Pick the resource you would like get access to from the Marketplace.
3. Fill in the access form.
  * If you are not the Principal Investigator of the organization, you will be required to submit it for approval first.
  * If you are - you can confirm the request at once.
  <img class="img-responsive center-block" src="/assets/img/guide/configure_order.png" width="700">
4. Once the request has been granted, you will be notified via email.
5. Granted resources and their usage will be visible in Project dashboard, Resources tab. 

### Getting support

1. Go to Project -> Resources tab.
2. Select the Resource for which you would like to get support.
3. Select "Issues" sub-tab and create a new request.
